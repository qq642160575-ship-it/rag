# api

本目录负责：对外 REST API 接口定义与请求调度。
不负责：具体的业务模型、向量化或存储实现（由更底层模块负责）。

## 文件说明

- main.py
  地位：API 服务总入口
  职责：初始化 FastAPI 实例，挂载路由，配置全局中间件与异常处理

- routes/
  地位：路由分发层
  职责：按业务维度划分接口（如 ingest, retrieval），负责请求参数校验与响应封装

- models/
  地位：API 数据模型层
  职责：定义基于 Pydantic 的请求（Request）与响应（Response）数据结构

> 声明：
> 一旦本目录结构或职责发生变化，请同步更新本文件

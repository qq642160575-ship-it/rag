# nodes

本目录负责：LangGraph 节点的具体动作实现
不负责：图拓扑结构的定义

> 声明：
> 一旦本目录结构或职责发生变化，请更新本文件

## 文件说明

- intent.py
  地位：入口节点逻辑
  职责：调用 LLM 分析用户意图

- expand.py
  地位：查询增强节点
  职责：根据意图进行关键词和问题扩展

- retrieval.py
  地位：检索中枢节点
  职责：执行向量检索并进行结果合并去重

- rerank.py
  地位：精排过滤节点
  职责：对检索到的文档进行二次打分排序

- judge.py
  地位：质量评估节点
  职责：判断当前检索到的文档是否足以回答问题

- rewrite.py
  地位：反思修正节点
  职责：在召回不足时重写查询语

- answer.py
  地位：终端生成节点
  职责：结合背景知识生成最终答案
